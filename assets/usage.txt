ldir [параметры] [<путь/к/директории>]

-d Обрабатывать также и поддиректории

-a Обрабатывать также и скрытые файлы и директории

-r Рекурсивно обрабатывать вложенные директории, это предполагает -d

-p Редактировать права доступа

-l Редактировать символьные ссылки

-S <правило> Сортировка файлов в списке. Правила сортировки:

name – сортровать по имени
type – сортировать по типу файла
crtime – сортировать по времени создания
atime – по времени доступа
mtime – по времени модификации
ctime – по времени изменения
